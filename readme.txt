To test a build :
1.You need to find the main folder in inspector where your project is located.
2.then open your terminal.
3.Type "cd" and then drag and drop your main folder into the terminal.
4.Now type "jekyll serve" to deploy this folder on your own local testing server.
5.Enjoy your changes in real time :) and just post to github when you are ready.


Edit everything in the main folder.
